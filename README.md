# 🛒 Smart Cart Customer Analytics

## 📌 Project Overview

Smart Cart Customer Analytics is a data analysis project that explores customer purchasing behavior and generates business insights using Python.

The project focuses on data cleaning, exploratory data analysis (EDA), visualization, and customer trend analysis to help understand shopping patterns and support data-driven decision-making.

---

## 🎯 Objectives

* Analyze customer data and purchasing behavior
* Perform data cleaning and preprocessing
* Explore customer demographics and trends
* Create meaningful visualizations
* Generate business insights from the dataset

---

## 📊 Dataset Features

The dataset contains customer-related information such as:

* Customer ID
* Age
* Gender
* Income
* Spending Behavior
* Purchase Patterns
* Other customer attributes

---

## 🔍 Exploratory Data Analysis (EDA)

The project includes:

* Dataset overview and summary statistics
* Missing value analysis
* Distribution analysis
* Customer segmentation insights
* Correlation analysis
* Data visualization using charts and graphs

---

## 🛠️ Technologies Used

* Python
* Pandas
* NumPy
* Matplotlib
* Seaborn
* Jupyter Notebook

---

## 📈 Key Insights

* Identified customer purchasing trends
* Analyzed demographic patterns
* Explored relationships between customer features
* Generated actionable business insights through visualizations

## 🚀 Getting Started

### Clone the Repository

```bash
git clone https://github.com/yourusername/Smart-Customer-Analytics.git
```

### Install Dependencies

```bash
pip install -r requirements.txt
```

### Run the Notebook

Open Jupyter Notebook and run:

```bash
jupyter notebook
```

Then open:

```text
Smart_Card.ipynb
```
